Attendance Button Flow
1
Student taps "Mark Attendance"
Single big button. No step selection. No confusion.
2
Browser requests GPS
Haversine check: must be within 151.75m of campus. If outside → blocked with error.
3
Device biometric fires
navigator.credentials.get() — browser shows fingerprint/Face ID. Only registered device works.
✓
Attendance confirmed
DB updates ABSENT → PRESENT. Timestamp + GPS distance logged.
Device Registration (First Time)
First login → prompt appears asking to register biometric device.

navigator.credentials.create()

Credential saved to user's devices[] array in DB. Only one device allowed. Second device attempt → blocked with message: "Device already registered. Contact HOD to reset."
